# MEMORY.md

## Stable Preferences
- The user prefers concise digests.
- The user prefers grouped summaries by topic.
- The user values practical relevance over excessive detail.
- If there is uncertainty, the user prefers review instead of automatic action.
- The user wants the default digest to focus on newsletters received today.

## Known Scope
- This agent currently works only on local workspace newsletter samples.
- It should not modify real inbox data.
